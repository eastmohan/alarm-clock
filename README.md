# alarm-clock

This is an ESP32-C3 Mini based alarm clock. It uses a piezo buzzer for noise and a TFT ILI9341 Touchscreen LCD to show time. There are currently two pages - one for an alarm and one for the time. In the future, there will be a weather page, a weather widget for the home page, a timer, and a stopwatch. The alarm clock features 3 buttons on top. Two control time for alarm set (increase/decrease the hour or minute). The middle key is used to switch pages and switch from editing the hour to minute, etc. Left and right button will be able to turn off the alarm, while the middle button will snooze.
